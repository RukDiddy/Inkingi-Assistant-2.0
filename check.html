INKINGI ASSISTANT — deployment self-test

The whole site is ~13 files. This page verifies every one of them, including the two screenshot bundles (images-corehr.zip: 117 pages, images-pip.zip: 73 pages).

How to fix
• Upload every file from the package into the same folder as index.html (GitHub web upload handles ~13 files in one commit without issues).
• Filenames are case-sensitive on GitHub Pages — keep them exactly as shipped.
• If you are viewing this locally via file://, zip checks are blocked by the browser — run python -m http.server 8080 in the folder and open http://localhost:8080/check.html.
